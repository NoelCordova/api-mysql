# Node API REST - MySQL

## Prerrequisitos:
* Node.js > v8
* MySQL >= 8.0.13

## Instrucciones:
1. Tener un servidor de MySQL ejecutado.
2. Crear una base de datos.
3. Crear un archivo llamado `config.js` dentro de la carpeta `config`. ej: `config-ejemplo.js`.
4. Ejecutar el comando 
```
npm run setup
```
